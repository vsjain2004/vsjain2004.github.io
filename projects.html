Projects

Computer Architecture

Processor Design for CprE 381 - Fall 2023

CprE 381 is a course on Computer Organization and Assembly level Programming focused on the basics of processor design and MIPS architecture. There were two major projects in the course during which three different processor designs were developed to fit a subset of the MIPS architecture. The projects are usually done in a team of 3 people, but I did all of these projects by myself. All designs were programmed in VHDL and were tested in Modelsim. The designs then went through a timing analysis in Quartus Prime based on a selected FPGA hardware. Learn more

Processor Design for CprE 281 - Fall 2022

CprE 281 is a course focused on the basics of digital logic design. For the Final Project, we are required to design a digital system for any purpose we want. This design is then run on Altera's DE2 FPGA Board. For my final project I designed a small-scale Single-Cycle Processor.

Robotics

Optical Recognition in PCBs for CprE 575 - Spring 2023

CprE 575 is a course focused on learning the basics of Computational Perception. The course starts with Image Processing and is followed by working on Videos. Then we deal with Audio analysis. For the Final project of the course, students work in groups to create something that incorporates at least one major concept from the course.

Embedded Sytems Project for CprE 288 - Spring 2023

CprE 288 is a course focused on the basics of Embedded Systems and Embedded Systems programming. It involves writing scripts in C to control a Roomba. The Roombas have been adapted with a baseboard that connect to a WiFi antenna and a Texas Instruments board. For the final project, groups of three to four people work to create a situation that can be simulated using Roomba and then program it to do so.

Software Design

Senior Design Project - Fall 2024 to Spring 2025

ComS 309: CyHawk Clash (Android App) - Fall 2023

ComS 309 is a course focused on learning about Android App Development using Java. Over the course of the semester, groups of 4 people develop an Android app. For the Forntend we used Android Studio, for the Backend we used SpringBoot and for our Database we used MySQL. Aside from basic Frontend and Backend code, we used advanced tools like Websockets as well as third-party APIs.
