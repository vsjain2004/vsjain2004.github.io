CyHawk Clash for ComS 309 - Fall 2023

Course Project | Software Development Practices

ComS 309 is a course on Software Development Practices, emphasizing Android app development in Java. For our project, my team and I developed an Android app using Android Studio for the frontend and Spring Boot for the backend, tested on tablets. Our team placed 2nd out of two course sections.

Project Details

Our team developed a 2D top-down shooting game as our project for ComS 309. The video on the right showcases all of the features we implemented by the end of the semester. View the code on Github.

As the Backend Lead, I focused on designing the database and implementing functionality to meet the frontend requirements. The backend team also structured the code to allow for scalability, making it adaptable for potential future features.

Skills and Knowledge Gained

This project reinforced my understanding of full-stack development, including integrating front-end requests with back-end services. I gained hands-on experience with database design and management, learning how to efficiently store and retrieve game data in a way that ensured quick access for gameplay. Collaborating on a large-scale codebase taught me the importance of clear documentation and code organization, while working on feature planning improved my strategic thinking and adaptability to project needs.
