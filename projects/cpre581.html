ISA Examination for CprE 5810 - Fall 2024

Course Project | Computer Systems Architecture

CprE 5810 explores the quantitative principles of computer architecture, covering instruction set design, processor organization, and memory hierarchy. For the semester project, my team analyzed and compared multiple Instruction Set Architectures (ISAs) based on the principles outlined in W.A. Wulf’s *“Compilers and Computer Architecture.”*

Project Details

In this project, we evaluated RISC and CISC ISAs, ranking them according to their adherence to the principles of regularity, orthogonality, and composability. We performed static binary analysis after compilation on selected benchmarks and, where applicable, used Gem5 simulations to measure CPI across architectures under identical processor configurations. My primary contribution involved setting up and running Gem5 simulations, ensuring consistent architectural parameters while managing runtime and memory constraints. A detailed project report can be viewed here.

Skills and Knowledge Gained

This project deepened my understanding of ISA evaluation and architectural trade-offs. I gained hands-on experience with Gem5 for performance analysis, learned to conduct static binary analysis, and developed a structured approach to comparing architectures based on quantitative design principles.
