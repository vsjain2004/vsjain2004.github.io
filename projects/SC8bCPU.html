Processor Design - Jan 2023 to Present

Extension to CprE 2810 Processor Design

Building upon the initial CPU built for the course project of CprE 2810, this project aims to enhance functionality via extension of its custom 8-bit minimal CISC ISA and later architectural enhancements. A version log starting from version 2 is available below.

CPU v3

  • Change and compress instruction opcodes to add new instructions
  • Add ability to add, subtract or swap two registers
  • Add ability to Swap memory address with a register
  • Add Logical, Arithmetic, and Circular Shift instructions
  • Add Register Decrement instruction
  • GitHub Code

CPU v2

  • Add Indirect and Indexed Memory Addressing Modes
  • Add 'Jump if Greater Than' and 'Jump if Greater than or Equal' instructions
  • Convert the Control system from a single unit to a Opcode Decoder + Control unit structure
