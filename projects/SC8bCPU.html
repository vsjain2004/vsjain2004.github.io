Processor Design - Jan 2023 to Present

Extension of the CprE 2810 Processor Design Project

Building upon the initial CPU developed for the CprE 2810 course project, this work enhances functionality through extensions to its custom 8-bit minimal CISC ISA and subsequent architectural improvements. A version log beginning with version 2 is provided below.

CPU v3

  • Redesigned and compressed instruction opcodes to enable additional instructions.
  • Added support for register-to-register arithmetic and swap operations.
  • Implemented memory-to-register swap functionality.
  • Introduced logical, arithmetic, and circular shift instructions.
  • Added a dedicated register decrement instruction.
  • View GitHub Code

CPU v2

  • Implemented indirect and indexed memory addressing modes.
  • Added conditional branching instructions: Jump if Greater Than and Jump if Greater Than or Equal.
  • Refactored the control logic from a single control unit into an opcode decoder + control unit architecture.
