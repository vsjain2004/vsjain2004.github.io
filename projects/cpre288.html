Embedded Systems Project for CprE 2880 - Spring 2023

Course Project | Embedded Systems I

CprE 2880 introduces the principles of embedded systems design and hardware-software interaction. For the semester project, my team programmed a TM4C123 microcontroller mounted on a Roomba to autonomously navigate a maze using an array of onboard sensors. A MATLAB-based GUI allowed manual control and real-time visualization of the Roomba's environment.

Project Details

For this project, my team and I programmed a TM4C123GH6PM microcontroller to solve a maze using multiple onboard and external sensors. The Roomba's front-mounted pressure sensors and a 180Â° rotating infrared (IR) sensor were used to detect obstacles, while a bottom-facing IR sensor prevented the robot from falling into pits. Communication between the microcontroller and the Roomba occurred via a UART interface, connected over Wi-Fi to our host machine. We developed a MATLAB GUI to visualize the maze, display sensor data, and control the Roomba's movement. As a fun extension, we also programmed the Roomba to play the Imperial March. The codebase is currently unavailable.

Skills and Knowledge Gained

This project provided hands-on experience with embedded systems programming and real-time sensor integration. I gained practical skills in configuring ADC, GPIO, timers, and PWM modules on the TM4C123 microcontroller, as well as interpreting and calibrating sensor outputs for reliable navigation. Additionally, I learned to design a responsive MATLAB GUI for human-machine interaction and developed an understanding of UART communication protocols and embedded control loops. These experiences deepened my appreciation for low-level hardware control and system debugging in embedded environments.
