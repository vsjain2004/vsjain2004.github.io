Processor Design for CprE 381 - Fall 2023

Course Project | Computer Organization and Assembly

CprE 381 is a course on Computer Organization and Assembly level programming, focusing on processor design and the MIPS architecture. During this course, I developed three processor designs to fit a subset of the MIPS architecture, all programmed in VHDL and tested in ModelSim. Timing analysis was performed in Quartus Prime, using a Harvard Architecture approach.

Project Details

Project 1: Single-Cycle Processor - I created a schematic, defined control signals, and assembled components, culminating in the final schematic design shown in the image. View the code on GitHub.

Project 2, Part 1: 5-Stage Pipelined Processor - Using the single-cycle processor as a base, I developed a software-scheduled, pipelined processor that relied on assembly program no-ops to bypass hazards. View the code on GitHub.

Project 2, Part 2: Hazard-Checking Processor - I implemented a hazard detection unit to address data clashes, adding new components and process statements to manage hazards. View the code on GitHub.

Through these projects, I gained hands-on experience with processor design, VHDL programming, and timing analysis, which has furthered my interest in processor design, particularly for machine learning applications.
